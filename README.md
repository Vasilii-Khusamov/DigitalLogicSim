# DigitalLogicSim

Симулятор цифровых логических схем на C# / .NET.

## Требования

- [.NET SDK 8.0](https://dotnet.microsoft.com/download) или выше

Проверка установки:

```bash
dotnet --version
```

## Запуск

Из корня репозитория:

```bash
dotnet run --project DigitalLogicSim
```

Или из папки проекта:

```bash
cd DigitalLogicSim
dotnet run
```

## Сборка

```bash
dotnet build
```

## Сборка Release

```bash
dotnet build -c Release
```
